## Town of Us: Mira has updated to [v1.4.0](https://github.com/AU-Avengers/TOU-Mira/releases/tag/v1.4.0)!

This update includes fixes for some critical bugs that were introduced in v1.4.0. There are also additional changes to settings, so be sure to check out the changelog!

This update also includes a download specifically for macOS and Linux users. Please be sure to do a fresh install.

See changelog below for bugfixes, changes and improvements.

<!--truncate-->

### Changelog

<details className="customdetails">
  <summary>**Role Bugfixes**</summary>

<details className="customdetails">
<summary>**Crewmates**</summary>
* Jailor no longer shows a kill screen upon execution.
* Seer no longer gains additional uses from unfinished player comparisons.
* Imitator can no longer become basic Impostor or Survivor.
* Vanilla Detective notes can be properly used in meetings ([PR #99](https://github.com/AU-Avengers/TOU-Mira/pull/99), Rewalo).
</details>

<details className="customdetails">
<summary>**Impostors**</summary>
* Fixed Eclipsal option name.
* Traitor can no longer spawn like a basic Impostor role.
* Double Undertakers can no longer grab the same body.
</details>

<details className="customdetails">
<summary>**Neutrals**</summary>
* Spectre now spawns consistently
</details>
</details>

<details className="customdetails">
  <summary>**Miscellaneous Bugfixes**</summary>

<details className="customdetails">
<summary>**Gamemode Fixes**</summary>
* Hide and Seek modifiers no longer appear in guessing menus.
* Classic Mode roles and modifiers can no longer spawn in Hide and Seek.
* All game keybinds are moved to the right-hand side, no longer blocking vent uses in Hide and Seek.
</details>

<details className="customdetails">
<summary>**Other Bug Fixes**</summary>
* Neutral Wildcard will no longer cause incorrect neutral role spawns.
* April Fools bugs no longer appear when joining a lobby.
* Fixed keybinds bypassing button checks from shields. 
</details>
</details>

<details className="customdetails">
  <summary>**Changes, Improvements and Additions**</summary>

<details className="customdetails">
<summary>**Changes**</summary>
* All intro blurbs have been improved and simplified.
* Crewpostor can no longer be a Jailor, Politician, Snitch, or Swapper.
</details>

<details className="customdetails">
<summary>**Improvements**</summary>
* Cached roles (such as Imitator or Traitor) will be displayed at the top of the Wiki.
* Stricter anticheat improvements ([PR #96](https://github.com/AU-Avengers/TOU-Mira/pull/96), Rewalo).
* Chat messages can now include most Unicode characters and can be up to 300 characters long ([PR #95](https://github.com/AU-Avengers/TOU-Mira/pull/95), Rewalo).
* The chat now displays secondary text bubbles for Lover, Jailor, Impostor, and Vampire chat.
* Lover chat is now a pink tint to make it stand out.
* Hunter has configurable task options.
* Engineer now has very configurable Vent, Fix, and Task settings.
</details>

<details className="customdetails">
<summary>**Additions**</summary>
* New Local Settings: Grenadier Flash Color & End Game Summary Message style.
* Better Polus Setting: Move Bathroom vent to Medbay.
* Better Mira HQ Setting: Better Vent Network.
* New Keybinds added for zooming in and out.
* New option: First Round Death Indicator.
* New command: **/roles** ([PR #95](https://github.com/AU-Avengers/TOU-Mira/pull/95), Rewalo).
* New Hide and Seek Modifier: Transporter.
</details>
</details>
