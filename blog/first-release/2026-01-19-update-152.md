## Town of Us: Mira has updated to [v1.5.2](https://github.com/AU-Avengers/TOU-Mira/releases/tag/v1.5.2)!

This update includes bug fixes and improved map configurations. Be sure to do a fresh install!

See changelog below for bugfixes, changes and improvements.

<!--truncate-->

### Changelog

<details className="customdetails">
  <summary>**Role Bugfixes**</summary>

<details className="customdetails">
<summary>**Ghost Roles**</summary>
* Spectre's role tab now displays the information properly.
* Ghostwalker roles should appear to spawn properly for all players.
* Spectator should no longer error out in the meeting.
</details>
</details>

<details className="customdetails">
  <summary>**Miscellaneous Bugfixes**</summary>
* Upper and Lower Deck are no longer broken visually on Submerged ([PR #117](https://github.com/AU-Avengers/TOU-Mira/pull/117), jukixyo).
* Cooldown Pause in Panels option is no longer duplicated in the settings menu.

</details>

<details className="customdetails">
  <summary>**Changes, Improvements and Additions**</summary>

<details className="customdetails">
<summary>**Changes**</summary>
* The Main Menu now displays the Discord Server and Source Code links next to the Fools Mode toggle.
</details>

<details className="customdetails">
<summary>**Improvements**</summary>
* Specific Better Maps settings can be changed Per-Map or via one setting.
* Camo Comms can be toggled for each map now.
</details>

<details className="customdetails">
<summary>**Additions**</summary>
Arsonist Option: Douses Possible at Once.
* Vanilla Tweak Option: Ghostwalkers can Fix Sabotages Toggle.
* Host Option: Multiplayer Freeplay Mode.
* Better Maps Options: Tasks to use Admin/Cams/Doorlogs/Vitals.
* Better Skeld Option: Quad-Group Vent System Toggle.
* Better Fungle Options: No Ladder Cooldown toggle.
* Better Submerged Options: Reactor/Oxygen Duration Changes.
* Better LI Options: Reactor/Oxygen/Mix-Up Duration Changes, No Ladder Cooldown Toggle.
</details>
</details>
