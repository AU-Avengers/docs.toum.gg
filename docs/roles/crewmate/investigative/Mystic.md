# Mystic

The Mystic is a Crewmate Investigative role that gets an alert when someone dies.

### Options

| Option | Description | Type | Default | Min/Max |
|----------|:-----------------:|:------:|:------:|:------:|
| Arrow Duration | How long a dead body arrow is shown for. | Seconds | 0.05s | 0s - 1s |
