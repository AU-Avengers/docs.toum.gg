# Install Guide

To install Town of Us: Mira, you must own the game on one of these platforms for each of their respective operating systems:

- **Windows:** Epic Games, Itch.io, Microsoft Store / Xbox **App**, or Steam.
- **Linux/Steam Deck:** Epic Games, Itch.io, or Steam.
- **Mac:** Epic Games or Steam.
- **Android:** Google Play via the Starlight mod launcher app.

Crossplay between the above platforms is supported, but all players must have the mod installed—**this is not a host-only mod**.
