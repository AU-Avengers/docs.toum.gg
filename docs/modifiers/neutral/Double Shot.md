# Double Shot

**Neutral Utility**

If you make an incorrect guess on a player during a meeting, you will get one second chance.

| Option             |                          Description                           |    Type    | Default |  Range   |
| ------------------ | :------------------------------------------------------------: | :--------: | :-----: | :------: |
| Double Shot Amount | Determines how many Neutral players can have the Double Shot modifier. |   Count    |    1    |  0 - 5   |
| Double Shot Chance |     Determines the chance of Double Shot being a modifier.     | Percentage |   50%   | 0 - 100% |
