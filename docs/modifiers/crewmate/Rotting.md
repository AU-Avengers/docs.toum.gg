# Rotting

#### Crewmate Postmortem

After a set amount of time, your body will rot away, preventing you from being reported

### Options

| Option | Description | Type | Default | Min/Max |
|----------|:-----------------:|:------:|:------:|:------:|
| Rot Delay | Time before the body rots away. | Seconds | 5s | 0s - 25s |
